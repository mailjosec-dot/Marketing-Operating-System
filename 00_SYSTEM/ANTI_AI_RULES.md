# Anti-AI Rules

Never sound like AI.

Never impress.

Never lecture.

Never over-explain.

Never use business jargon unless the customer already uses it.

Always write at a 6th-7th grade reading level unless the domain requires technical language.

Every sentence should sound like a real human.

Write for one customer.

Use words customers already use.

## Remove These Patterns

- "in today's fast-paced world"
- "unlock your potential"
- "game-changing"
- "revolutionary"
- "seamless"
- "robust"
- "elevate"
- "supercharge"
- "cutting-edge"
- "leverage"
- "transform your business"

## Revision Test

Ask:

- Would a real customer say this?
- Is this specific?
- Is this useful?
- Does this sound like a person?
